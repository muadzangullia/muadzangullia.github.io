02 — WORK

Work

Projects I've built, from internship to school.

Dragino IoT Sensor Architecture
SMRT · Ongoing

Redesigned SMRT's train temperature-sensor system to cut cost. Flow: Sensor → VPN → Azure IoT → ADX → Grafana/Power BI.

My part

Designed the architecture, configured the sensors, built the Grafana dashboard.

Result

Real-time temperature visibility for engineers, at lower cost than the old system.

Challenge

An experimental, undocumented Dragino sensor kept failing to connect over MQTT. I isolated the cause by elimination — network, broker, then auth — and traced it to a firmware password-length limit. Dragino patched it once I flagged it.

SAP → SQL Defect Data Pipeline
SMRT · Completed

Pipeline that cleans and reshapes messy SAP defect data into structured SQL tables engineers can query and visualise.

My part

Built it end-to-end with Azure Data Factory + PySpark — fixing inconsistent formats, duplicates, and missing fields, then classifying and grouping defects into categories so they can be charted and filtered.

Result

Clean, categorised tables engineers query daily — and feed straight into Power BI for defect trend analysis, instead of manually cleaning SAP exports.

Cold-Chain Temperature Monitoring
School · IoT Module

The same problem as Dragino, tackled solo on a different stack — real-time temperature monitoring for cold-chain medication, built end-to-end on AWS.

My part

Built the full stack solo across 6 AWS services: IoT Core for ingestion, Lambda for processing, DynamoDB + S3 for storage, SNS for alerts, CloudWatch for the dashboard.

Result

Flags temperature breaches in transit before medication is compromised — Dragino's core idea, rebuilt on AWS.
