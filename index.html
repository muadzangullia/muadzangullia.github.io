02 — DIGITAL EVIDENCE

Artefacts from the job

Two pieces of work that map directly onto my role as a Data Engineer — one in progress, one shipped and in use.

Dragino IoT Sensor Architecture
Ongoing

A redesigned temperature-sensing architecture for train maintenance, built to replace an older, more expensive system. Sensor data flows through a VPN network into Azure IoT, lands in Azure Data Explorer (ADX), and is visualised in Grafana and Power BI.

My contribution

Ideated the end-to-end architecture, configured the physical sensors, and built a personalised Grafana dashboard from scratch to visualise incoming readings.

How it's used

Once live, the dashboard gives maintenance engineers real-time visibility into train temperature conditions — supporting preventive maintenance decisions at a lower hardware and operating cost than the legacy sensor system.

Challenge — experimental hardware, no documentation

The sensor unit was an unreleased, experimental device supplied directly by the Dragino team, so there was no public documentation to fall back on. It repeatedly failed to connect via MQTT with no obvious cause.

I worked through the problem systematically — ruling out the VPN/network setup, broker configuration, and authentication one by one — and traced it to the device firmware rejecting passwords beyond a certain length. Flagging this to Dragino led them to patch their firmware to accept longer inputs.

SAP → SQL Defect Data Pipeline
Completed

An automated pipeline that extracts train defect data from SAP and transforms it into clean, structured SQL tables — replacing manual, error-prone handling of raw exports.

My contribution

Built the pipeline end-to-end using Azure Data Factory for orchestration and PySpark in Synapse Notebooks for the cleaning logic — handling inconsistent formats, duplicate records, and missing fields.

How it's used

Maintenance engineers query the resulting SQL tables directly for day-to-day defect inspection, instead of manually reconciling messy SAP exports — cutting down the manual effort involved in preparing data for review.
